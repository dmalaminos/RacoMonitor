Acerca de...

RacóMonitor es una extensión para Chrome que gestiona los avisos del Racó de la FIB (UPC) y lanza notificaciones cuando se publican nuevos avisos. Pretende mejorar la eficiencia del usuario facilitando accesos rápidos a los avisos, que pueden ser ordenados según diferentes criterios. Gracias a las notificaciones, que avisan casi instantáneamente cuando se publican nuevos avisos, también le dará un descanso a la tecla F5.

El núcleo de la aplicación está programado en JavaScript, con las librerías jQuery y jsOAuth. Además, también utiliza la librería Moment.js y Bootstrap para la interfaz de usuario.



Instalación

Para empezar, necesitarás descargar la extensión (.crx), que puedes encontrar comprimida en este enlace.

A partir de finales de mayo de 2014, Google impuso a Chrome para Windows que no pudiera instalar extensiones locales, descargadas fuera de la Web Store; aún así, todavía es posible instalar una extensión de forma local activando el "modo de desarrollador", como se explica más abajo. Esta limitación por ahora no afecta a usuarios de otros sistemas operativos.

Los dos primeros métodos representan la manera más sencilla de instalar RacóMonitor en Windows y Linux, respectivamente. El segundo también debería de funcionar para Mac. El tercer método es una alternativa que funciona para cualquier SO.

Método #1 (recomendado)

  1. Abre una nueva pestaña en Chrome con la dirección “chrome://extensions/” (también se puede acceder desde Herramientas>Extensiones)
  2. Asegúrate que esté activada la casilla “Modo de desarrollador”, en el margen superior
  3. Arrastra el fichero RacoMonitor.crx dentro de la navegador
  4. Se abrirá una nueva pestaña mostrando la página habitual de login del Racó, si tu sesión de usuario no estaba previamente abierta. Aparecerá una pantalla pidiendo permiso para utilizar la API interna. De todos los datos que informa que se podrán acceder, RacoMonitor sólo accederá a los avisos más recientes
  5. Una vez se ha dado permiso, aparecerá un mensaje. Para terminar el proceso de autorización es importante que cierres esta pestaña

Método #2 (recomendado para Linux)

  1. Abre una nueva pestaña en Chrome con la dirección “chrome://extensions/” (también se puede acceder desde Herramientas>Extensiones)
  2. Arrastra el fichero RacoMonitor.crx dentro de la navegador
  3. Se abrirá una nueva pestaña mostrando la página habitual de login del Racó, si tu sesión de usuario no estaba previamente abierta. Aparecerá una pantalla pidiendo permiso para utilizar la API interna. De todos los datos que informa que se podrán acceder, RacoMonitor sólo accederá a los avisos más recientes
  4. Una vez se ha dado permiso, aparecerá un mensaje. Para terminar el proceso de autorización es importante que cierres esta pestaña

Método #3

  1. Descomprime el archivo RacoMonitor.crx en un directorio de tu sistema; ten en cuenta que el directorio y sus contenidos deberán de permanecer ahí mientras quieras utilizar la extensión
  2. Abre una nueva pestaña en Chrome con la dirección “chrome://extensions/” (también se puede acceder desde Herramientas>Extensiones)
  3. Asegúrate que esté activado el checkbox “Modo de desarrollador”, en el margen superior
  4. Utiliza la opción para cargar la extensión descomprimida en el directorio del paso 1
  5. Se abrirá una nueva pestaña mostrando la página habitual de login del Racó, si tu sesión de usuario no estaba previamente abierta. Aparecerá una pantalla pidiendo permiso para utilizar la API interna. De todos los datos que informa que se podrán acceder, RacoMonitor sólo accederá a los avisos más recientes
  6. Una vez se ha dado permiso, aparecerá un mensaje. Para terminar el proceso de autorización es importante que cierres esta pestaña


Uso

Una vez instalado correctamente, se añadirá un icono con el logo de la FIB en la toolbar. Haciendo clic sobre éste aparecerá un popup con tres pestañas: últimos avisos (avisos no leídos), avisos anteriores (avisos leídos) y configuración.

  • Últimos avisos: aquí aparecerán los avisos que no hayas leído. También aparecerá una barra superior con un acceso directo al Racó, un botón para abrir todos los avisos (y en el caso de no estar leídos, marcarlos como tal), y un desplegable con opciones para ordenar los avisos. Además, hay un botón para refrescar, aunque cada vez que se abre el popup ya se comprueba automáticamente si se han publicado nuevos avisos.
  • Avisos anteriores: aquí aparecerán los avisos que hayan sido marcados como leídos, junto con la misma barra superior que en la pestaña anterior.
  • Configuración: presenta las opciones personalizables de la extensión.
    • Frecuencia de refresco: cada cuánto tiempo comprueba si hay nuevos avisos, entre un minuto y una hora.
    • Formato de la fecha: relativo (p.e. hace 2 horas) o absoluto (p.e. miércoles, 19 de mayo de 2014, 23:49:01).
    • Notificaciones de escritorio: si el navegador debe lanzar una notificación cuando haya llegado un nuevo aviso.
    • Sonido de notificación: si debe reproducir un sonido cuando haya llegado un nuevo aviso.
    • Idioma: a elegir entre castellano y catalán.


FAQ (Preguntas frecuentes)

¿Qué versión de RacóMonitor tengo instalada?

Abre una nueva pestaña en Chrome con la dirección “chrome://extensions/” (también se puede acceder desde Herramientas>Extensiones). El número de versión aparecerá junto al título de RacóMonitor.

¿Cómo puedo cambiar el sonido de notificación?

Si instalaste la extensión mediante el método #3, puedes ir a la carpeta de RacóMonitor que descomprimiste, entrar al directorio sounds y reemplazar el archivo alarm.mp3 por el sonido que quieras. Si utilizaste otro método, deberás de extraer el contenido de RacoMonitor.crx en una carpeta, substituir el archivo de sonido como se indicaba antes y volver a comprimir la carpeta con el formato crx. Para hacer esto último, puedes ir a la página de extensiones (“chrome://extensions/”) y utilizar la opción Empaquetar extensión.

¿Cómo accede la extensión a mi perfil del Racó?

RacóMonitor no accede a tu perfil directamente, sino que usa la API privada del Racó, con la que el usuario da permiso durante el proceso de autorización. En ese momento, una pantalla informa que la extensión podrá acceder a avisos, horario, calendario... pero realmente sólo se utilizan los avisos más recientes, concretamente el archivo /api-v1/avisos.rss. Puedes encontrar más información sobre la API del Racó aquí.

¿Cómo puedo revocar el acceso a la API del Racó para una aplicación?

Dirígete a esta página, donde podrás ver a qué aplicaciones has dado permisos y revocarlos si es necesario.

¿Por qué me llegan avisos repetidos —con el mismo título?

Esto ocurre porque los profesores han actualizado el aviso añadiendo, modificando o eliminando contenido. Aunque no suelen cambiar el título en estas situaciones, la extensión detecta que el contenido es diferente y lo muestra como un nuevo aviso con el mismo título pero con la fecha de publicación modificada. A algunos profes les gusta ir actualizando el aviso a medida que lo van escribiendo, hay que tener un poco de paciencia :)

¿Por qué la extensión requiere permiso para acceder a Dropbox?

Chrome informa que la extensión requiere permisos para "acceder a tus datos en www.dropbox.com" porque RacóMonitor necesita comprobar periódicamente un archivo de texto alojado en Dropbox, que servirá para avisar cuando haya nuevas versiones. Realmente nunca va a acceder a tus datos personales, pero necesita este permiso para acceder al dominio de Dropbox.

¿Por qué debería de borrar los datos de avisos anteriores al empezar un nuevo cuatrimestre?

La extensión está ideada para que trabaje con los avisos de asignaturas de un solo cuatrimestre. Si no se hace así es posible que siga funcionando, pero su comportamiento no ha sido testado. Además, los links al Racó de avisos de otro cuatri habrán dejado de funcionar, por lo que no tiene mucha utilidad conservar los avisos en memoria. Puedes borrarlos usando el botón "Borrar datos" en la pestaña de Configuración.
